#  To-Do List App - Backend

##  Description
This is the backend of the To-Do List application built using Node.js, Express.js, and MongoDB.  
It provides RESTful APIs for managing tasks: create, read, update, delete, toggle status, and search.



##  Tech Stack
- Node.js
- Express.js
- MongoDB (Mongoose)
- dotenv
- cors


##  Setup Instructions

## 1. Install dependencies

cd backend

npm install

npm run dev
