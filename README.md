#  To-Do List App (Assignment 8)

##  Description
This is a full-stack To-Do List application built using Node.js, Express.js, MongoDB, and React.  
The application allows users to manage daily tasks with full CRUD operations and real-time UI updates.

---

## Features
- Add new task
-  Delete task
-  Update task
-  Mark task as completed (Toggle status)
- Search tasks
-  Responsive UI

---

##  Tech Stack
### Frontend:
